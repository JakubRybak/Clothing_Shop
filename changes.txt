4. ANIMATION OF ADDING TO CART (ICON & BUTTON)
Description:
- Navbar Cart Icon: When the cart is updated (via HTMX), the cart container pulses (scales up and glows green).
- Add to Cart Button: On the product detail page, clicking the button shows a "Adding..." spinner, and on success turns green with "Added to Cart!" text before reverting.

Code Snippet (templates/base.html - Script):
```html
    <script>
        document.body.addEventListener('htmx:afterSwap', function(event) {
            // Check if the swapped content is the menu-cart
            if (event.detail.target.id === 'menu-cart') {
                var cartContainer = document.getElementById('menu-cart');
                cartContainer.classList.add('cart-updated');
                
                // Remove class after animation to allow re-triggering
                setTimeout(function() {
                    cartContainer.classList.remove('cart-updated');
                }, 400); // Matches animation duration
            }
        });
    </script>
```

Code Snippet (static/css/styles.css - Animation Keyframes):
```css
/* Animation for Cart Update */
@keyframes cart-pulse {
    0% { transform: scale(1); color: rgba(255,255,255,0.55); } /* standard nav-link color */
    50% { transform: scale(1.4); color: #198754; text-shadow: 0 0 5px #198754; } /* Success green with glow */
    100% { transform: scale(1); color: rgba(255,255,255,0.55); }
}

.cart-updated {
    animation: cart-pulse 0.5s cubic-bezier(0.175, 0.885, 0.32, 1.275); /* bouncy effect */
    display: inline-block; /* Ensure it respects transform */
}
```

Code Snippet (static/js/product_detail.js - Button Logic):
```javascript
// Add-to-Cart Button Animation Logic
document.addEventListener('DOMContentLoaded', function() {
    const addToCartBtn = document.getElementById('add-to-cart-btn');
    
    if (addToCartBtn) {
        // Store original text
        let originalText = addToCartBtn.innerHTML;

        // Ensure smooth color transition
        addToCartBtn.style.transition = 'all 0.3s ease';

        addToCartBtn.addEventListener('htmx:configRequest', function() {
            // Loading State
            addToCartBtn.innerHTML = '<span class="spinner-border spinner-border-sm me-2" role="status" aria-hidden="true"></span>Adding...';
            addToCartBtn.disabled = true;
        });

        addToCartBtn.addEventListener('htmx:afterOnLoad', function(evt) {
            if (evt.detail.xhr.status === 200) {
                // Success State
                addToCartBtn.classList.remove('btn-primary');
                addToCartBtn.classList.add('btn-success');
                addToCartBtn.innerHTML = 'Added to Cart! <span class="fw-bold">✓</span>';
                
                // Revert after 2 seconds
                setTimeout(() => {
                    addToCartBtn.classList.remove('btn-success');
                    addToCartBtn.classList.add('btn-primary');
                    addToCartBtn.innerHTML = originalText;
                    addToCartBtn.disabled = false;
                }, 2000);
            } else {
                // Error State (optional, revert immediately)
                addToCartBtn.innerHTML = originalText;
                addToCartBtn.disabled = false;
            }
        });
    }
});
```

5. PROMPT BAR & FILTER LOGIC (BADGES & REMOVAL)
Description:
- Badges: Filters that are set automatically by the AI (from the prompt) are distinguished from manual selections. They get a special "✨" badge if they were purely AI-detected.
- Removal Logic: When the user removes a prompt (via the 'x' button), only the filters associated with that prompt (AI filters) are cleared. Filters manually checked by the user remain active because their inputs are not removed from the DOM.

Code Snippet (store/templates/store/partials/filter_sidebar_content.html - Badge Display):

```html
        <!-- Color Filter -->
```

Code Snippet (templates/base.html - Removal Logic):
```javascript
    function removeQuery(btn) {
        // Show spinner, hide 'x'
        btn.querySelector('.delete-icon').classList.add('d-none');
        btn.querySelector('.spinner-border').classList.remove('d-none');
        btn.disabled = true;

        // Find the hidden input in this group and remove it so it's not sent
        var group = btn.closest('.input-group');
        var input = group.querySelector('input[name="q"]');
        if (input) input.remove();

        // Use htmx to trigger the form submission
        var form = document.getElementById('searchForm');
        htmx.trigger(form, 'submit');
    }
```
